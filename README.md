# Job Tracker Pro

A modern Full Stack job application tracker designed to help job seekers manage applications, interviews and recruitment activities.

## Features

- Dashboard with statistics
- Create job applications
- Edit job applications
- Delete job applications
- Application status tracking
- PostgreSQL database persistence

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL
- Supabase

## Roadmap

- Authentication
- Search and filtering
- Kanban board
- Interview management
- Calendar integration
- AI-powered assistance
